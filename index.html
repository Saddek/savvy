Savvy

An easy-to-use framework for HTML-based app development (tablet, smart phone, smart TV or web).

View project onGitHub

Savvy is an easy-to-use framework for developing single-page HTML5 applications. It is intended for development of tablet, smart phone and smart TV apps, but it can also be used on PC and the web.

Savvy emphasises HTML, CSS and JavaScript as distinct technologies and employs a Model-View-Presenter-style paradigm for app development. Alongside HTML, CSS and JavaScript, Savvy understands JSON (and JXON) data models right out of the box.

Documentation

Savvy apps are composed of screens. Screens in turn are composed of HTML, CSS and JavaScript snippets. Savvy provides the mechanism for transitioning between screens.

Since Savvy solely provides a framework on the composing and transitioning between screens, it completely agnostic towards the choice of other libraries or frameworks. So, a developer may choose, for example, to perform DOM manipulation Vanilla JS, jQuery or Prototype.

Full documentation is available on the Savvy GitHub wiki.
