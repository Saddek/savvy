Savvy

An easy-to-use framework for HTML-based apps (for tablet, smart phone, and smart TV).

Savvy is an easy-to-use framework for developing single-page HTML5 applications. It is intended for development of tablet, smart phone and smart TV apps.

Savvy employs a modularised Model-View-Presenter-style paradigm for app development. Alongside HTML, CSS and JavaScript, Savvy understands JSON data models right out of the box.

Overview

Savvy apps are composed of screens. Screens in turn are composed of HTML, CSS and JavaScript snippets. Savvy provides the mechanism for transitioning between screens.

Since Savvy sole function is to provide a framework on the composing and transitioning screens, it is agnostic towards the choice of other libraries or frameworks. So, a developer may choose to perform DOM manipulation using Vanilla JS, jQuery or Prototype, for example. Or may choose to use Handlebars, Knockout or Dush for templating.
