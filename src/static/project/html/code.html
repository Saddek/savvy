Design it your way

Savvy is very clear about what it does: Savvy is an MVP application framework. It is the rock upon which you build your UI and presentation logic. And build it your way.

We're not about themes. Savvy apps are bespoke creations. Each one looks and behaves differently. What Savvy gives you is first-line-support for most the flexible UI paradigms and a laser-sighted MVP framework to develop on. That's the canvas. What goes in it is your creation.

Presentation first

Savvy apps are built on a bedrock of a modular HTML5 and CSS3 presentation layers backed highly-optomised JS presenter code.

HTML5 is an exciting new direction for cross-platform development combining a suite of declaratively programmed and natively-implemented widget types – from text and forms to images, audio and video and interactive 2D and 3D drawing surfaces.

Combined with the rich presentation language that is CSS3 and first-line-support for most the flexible UI paradigms, HTML5 in Savvy delivers a powerful view onto any application: be it the world's most functional admin interface or the next immersive gaming experience.

The MVP architecture beneath any Savvy app then backs this up with as-needed support from your non-native code.

Hybrid apps

The Savvy framework is built for fire-and-forget integration with Apache Cordova, the leading hybrid application framework for mobile and tablet. Developing for hybrid gives your app the opportunity mobilise out from HTML5 (should you need to) and harness the native potential of whatever device your application is running on.

And with Adobe PhoneGap Build, you don't even need to install any SDKs. You can build app-store ready hybrid apps in the cloud seamlessly from Savvy.

Choose your language!

Don't like JavaScript? No problem. Whatever your taste in language, Savvy comes pre-packed with support for four major coding paradigms. Keep it loose and imperative with CoffeeScript or try strictly-typed and classical object orientation with TypeScript. Planning on pushing things to the limit? Code like a god with C or C++. Or hack out your next great adventure in everyday JavaScript.

No matter the language you choose, Savvy compiles it down to highly-optomized JS using the JavaScript as an assembly language perspective.

And CSS? Savvy has that covered too. Write in vanilla CSS – or go up a gear with Sass or Less. No matter what, Savvy builds and optomised your CSS before is passes out the door. In fact, Savvy apps are so optomised, we even optomise your HTML!

What about other frameworks?

Use them! Savvy is not just agnostic, it was designed from the ground up to leverage the very best of the modern development ecosystem.

Use jQuery, or Prototype, or code vanilla JS. Add bindings with Veu.js, Rivets.js, or Knockout. Create your own UI or start with Foundation or Bootstrap – or the mobile-specific Ratchet framework.

Then sprinkle you application with just-add-water animations from Animate.css, draw icons from FontAwesome and the Noun Project and patterns from Subtle Patterns, and present your app with a killer font from Google Fonts. Savvy is just the beginning of your journey – the first step in the right direction.
